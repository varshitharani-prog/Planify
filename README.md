## 🚀 Overview

Planify is a responsive task management application designed to help users organize, track, and complete tasks efficiently. It provides an intuitive workflow for managing tasks, monitoring progress, and improving productivity through an integrated AI Productivity Assistant powered by Google's Gemini API.

## ✨ Features

### ✅ Task Management

* Create, edit, and delete tasks
* Assign priority levels (High, Medium, Low)
* Set due dates for tasks
* Organize tasks across:

  * Todo
  * In Progress
  * Done

### 🎯 Drag and Drop Workflow

* Move tasks between different stages seamlessly
* Interactive task organization using DnD Kit

### 📊 Progress Dashboard

* View task statistics
* Track completion percentage
* Visual progress bar

### 🤖 AI Productivity Assistant

* Generate study plans
* Get productivity suggestions
* Receive interview preparation guidance
* Obtain learning roadmaps
* Powered by Gemini API

### 💾 Local Storage Persistence

* Saves tasks automatically
* Retains data after page refresh

### 📱 Responsive Design

* Mobile-friendly interface
* Hamburger navigation menu
* Optimized for desktop and mobile devices

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Planify.git
cd Planify
```

### Frontend Setup

```bash
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
```

Create `.env`:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run backend:

```bash
node server.js
```
